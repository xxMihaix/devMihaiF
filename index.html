Languages:

  • HTML

    It is the foundational language used to structure the content of a web page, defining elements such as headings, text, images, and links.

  • CSS

    It allows the styling of HTML elements and controls the visual appearance of the website, including colors, fonts, layout, and graphic effects.

  • JavaScript

    A programming language that adds interactivity to web pages, enabling dynamic actions and real-time content updates.

  • Node.js

    A runtime environment that allows JavaScript to run on the server side, enabling fast, scalable, and efficient backend development.

  • Express.js

    A lightweight web framework for Node.js that simplifies building APIs and server-side applications with powerful routing and middleware features.

  • MongoDB

    A NoSQL database that stores data in flexible, JSON-like documents, making it easy to handle large, unstructured, and evolving datasets.

  • PHP

    A server-side language used for generating dynamic content and interacting with databases in web applications.

  • MySql

    A relational database management system used for efficiently storing and organizing a website’s data.
